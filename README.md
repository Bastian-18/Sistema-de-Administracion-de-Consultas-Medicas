# SistemaClinico

Sistema de gestión clínica desarrollado con Java y OpenXava.

## Descripción

Proyecto base para la gestión de una clínica, incluyendo el registro de clientes, doctores, consultas y tratamientos.

## Estructura del Proyecto

```
src/main/java/
  └── clinica/
      ├── Cliente.java
      ├── Doctor.java
      ├── Consulta.java
      ├── Tratamiento.java
      └── SistemaClinico.java (clase principal)
```

## Estado del Proyecto

Proyecto en fase inicial. Estructura base preparada para comenzar el desarrollo.
